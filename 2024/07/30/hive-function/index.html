一些常用的 Hive 时间日期函数

发表于 2024-07-30 分类于 基础知识整理 Waline： 本文字数： 481 阅读时长 ≈ 2 分钟

参考文档：

  • Flink时间日期函数

  • HiveUDFs-DateFunctions

日期和时间的提取

  • CURRENT_DATE: 返回当前日期。

  • CURRENT_TIMESTAMP: 返回当前时间戳。

  • UNIX_TIMESTAMP([string date]): 将日期字符串转换为Unix时间戳（秒数）。

  • FROM_UNIXTIME(int): 将Unix时间戳转换为日期字符串。

  • TO_DATE(string timestamp): 将时间戳字符串转换为日期。

日期的格式化和转换

  • DATE_FORMAT(date/timestamp/string ts, string fmt): 将日期、时间戳或日期字符串格式化为指定的字符串格式。

  • CAST(expression AS date): 将表达式转换为日期类型。

  • TO_DATE(string timestamp): 将时间戳字符串转换为日期。

  • Flink: TO_TIMESTAMP(string date)将日期字符串转换为时间戳。

日期计算

  • DATE_ADD(date/timestamp startdate, int days): 返回指定日期加上若干天后的日期。

  • DATE_SUB(date/timestamp startdate, int days): 返回指定日期减去若干天后的日期。

  • DATEDIFF(string enddate, string startdate): 返回两个日期之间的天数差。

日期部分提取

  • YEAR(date/timestamp/string): 返回日期的年份部分。

  • MONTH(date/timestamp/string): 返回日期的月份部分。

  • DAY(date/timestamp/string): 返回日期的天数部分。

  • HOUR(date/timestamp/string): 返回时间的小时部分。

  • MINUTE(date/timestamp/string): 返回时间的分钟部分。

  • SECOND(date/timestamp/string): 返回时间的秒数部分。
